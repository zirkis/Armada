# Armada
Manage your fleet from a touch.
Autonomous vehicle fleet management simulation, coded in JS with the Ember framework.
By HAOUI Benjamin & DIDELOT Kevin. 

## Installation

* `git clone <repository-url>` this repository
* `cd client`
* `yarn install` (or `npm`)
* `cd ../server`
* `yarn install` (or `npm`)

## Run

* `cd client`
* `yarn start`
* `cd ../server`
* `yarn start`
